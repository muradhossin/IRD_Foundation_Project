## Setup

Clone the repository

```bash
  git clone https://github.com/muradhossin/IRD_Foundation_Project.git
```
Move to the desired folder

```bash
  cd \IRD_Foundation_Project.git
```
To run the app, simply write

```bash
  flutter pub get
```
```bash
  flutter run
```
